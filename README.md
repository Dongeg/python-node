### String

下面几种都能表示字符串，三引号可以换行

'hello world' 

"hello world" 

'''
hello
world
'''

"""
hello
world
"""

字符串前面加r,表示原始字符串，里面的转义符将不被转译

>>> print(r'hello/nworld')

hello/nworld

#### 字符串操作

字符串可以 + * 

>>>'1' * 3

111

>>>'123'[0]

1

>>>'123'[-1]

>>> 'helloworld'[0:3]
